How this demo is built

01

Form capture

Property details are captured client-side and validated before submission.

02

Claude API

A Netlify function sends the structured brief to Claude, prompted to reason like a property appraiser.

03

Admin panel

The admin types plain-English instructions — Claude regenerates the report following them, in context.

04

Report render

The response is parsed into a structured, ledger-style report in the browser.

Site & Sum is a demo project built to showcase an AI-assisted appraisal workflow — Claude API + Netlify. Not a real valuation service.
